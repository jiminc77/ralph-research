# Title

Where to Act versus Which Way to Move: A Preregistered Factorial Pilot on One-Step Deformable Linear Object Manipulation

# Abstract

Hybrid contact actions $a=(p,u)$ couple a decision about *where* to act (contact location $p$) with a decision about *how* to move (motion parameter $u$). Methods that jointly select both factors do not reveal which factor carries the achieved one-step progress. We report a preregistered factorial intervention study on quasi-static deformable linear object (DLO) manipulation in a GPU-parallel rod simulator: for each canonicalized state we execute the complete $8\times8$ table of contact arc-length positions versus fixed-magnitude planar motion directions and decompose the resulting one-step progress into contact, direction, and interaction contributions. We deliberately scope the claim narrowly: the comparison is contact-location choice versus fixed-magnitude planar-direction choice, not a universal "where versus how," because the two factors have asymmetric expressiveness. On a pilot of 40 states across three tasks (2,560 executed action cells, candidate C0; preregistered scale was not reachable in the available budget), the primary contrast $\Delta_{WH}=\mathbb{E}_s[v_P-v_U]$ is +0.041 [+0.029, +0.053] for straightening, -0.034 [-0.073, +0.005] for single-bend shaping, and -0.130 [-0.161, -0.100] for endpoint repositioning (simultaneous 95% intervals; minimum preregistered effect 0.02). Under the preregistered labeling rule, straightening is contact-dominant, single-bend shaping is balanced/inconclusive, and endpoint repositioning is direction-dominant on this pilot. Repeatability, drift, and noise-floor controls pass on all used tables, but a canonicalization re-placement artifact invalidates 42/192 state tables (quarantined and reported separately) and makes the strict preregistered invalid-rate guard unsatisfiable at pilot scale, so no candidate regime is formally selectable and all numbers are descriptive. We release estimand definitions, guard thresholds, and per-cell raw tables; every number is pilot-scale evidence, not a confirmatory finding.
